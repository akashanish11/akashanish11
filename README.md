🧳 Featured Expeditions (Projects)
<details open>
<summary><b>🔐 VulnSpectra — Smart Contract Vulnerability Detection</b> <code>IEEE ICSCST 2026</code></summary>
<br/>
A hybrid CodeT5+ and RGAT framework that constructs heterogeneous program graphs (CFG + DFG) from Solidity contracts and classifies vulnerabilities using relation-aware attention and graph-level pooling.
Metric	Detail
Stack	CodeT5+, RGAT, PyTorch, PyTorch Geometric, Python
Scale	~10,000 Ethereum smart contracts
Performance	95.19% F1-score · 96.07% accuracy
Security	100% recall on timestamp dependency & dangerous delegatecall classes
Impact	11% F1 gain over best LLM-graph hybrid baseline
Repository	View on GitHub
Designed and implemented the full pipeline as first author — from framework architecture and PyTorch implementation to conference presentation — co-developed equally with a research partner.
</details>
<details>
<summary><b>🔗 Certificate Validation System using Blockchain</b> <code>Team of 4</code></summary>
<br/>
A decentralized certificate verification system that eliminates forgery by anchoring certificate hashes on-chain while keeping storage costs minimal.
Metric	Detail
Stack	Ethereum, Solidity, IPFS, Web3.py, Python, Streamlit, MetaMask
Scale	Full issuance-to-verification lifecycle
Performance	Gas-optimized contract logic
Security	Only hash on-chain — original files never exposed
Impact	Instant public verification, automatic forged-certificate rejection
Repository	View on GitHub
Built the Python/Web3.py backend to hash certificates at issuance and register them on-chain via MetaMask-signed transactions, with a Streamlit portal enabling instant on-chain verification.
</details>
<details>
<summary><b>🛒 SD Solutions Deck — Computer Hardware E-Commerce Platform</b> <code>Team of 2 · Client Project</code></summary>
<br/>
A full e-commerce platform for a paying client, featuring a custom PC configurator with real-time compatibility validation.
Metric	Detail
Stack	PHP, MySQL, HTML, CSS, JavaScript
Scale	Products, orders, inventory, service bookings
Performance	Session-based multi-step checkout flow
Security	Server-side cart & compatibility validation
Impact	Automated stock deduction, low-inventory admin alerts
Repository	View on GitHub
Designed the relational MySQL schema and built PHP modules covering the complete order lifecycle for a live commercial client.
</details>
<details>
<summary><b>🏟️ Arenagrounds — Online Turf Booking Application</b> <code>Team of 2 · Android</code></summary>

Scale	18 hourly slots/turf/day across 5 cities
Performance	Real-time Firestore stream-based slot sync
Security	UID-scoped Firestore security rules
